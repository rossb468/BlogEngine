# Getting Started with Swift

Swift is a powerful and intuitive programming language developed by **Apple** for building apps across the entire Apple ecosystem. Since its introduction in 2014, Swift has grown into a mature language with a passionate open-source community. Whether you are building a sleek *iOS* app, a responsive *macOS* utility, or a server-side backend with Vapor, Swift offers the safety of modern type checking with the performance of a compiled language.

![Swift code in an Xcode playground](https://picsum.photos/seed/swift-playground/800/400)

One of Swift's greatest strengths is its readability. Variables are declared with `let` and `var`, optionals make null safety explicit, and powerful features like pattern matching and generics keep your code both concise and expressive. The Swift Package Manager has also matured significantly, making dependency management straightforward for projects of any size.

If you are just getting started, download [Xcode](https://developer.apple.com/xcode/) and create a new Playground. Playgrounds let you experiment with Swift in real time -- type a line of code and see the result immediately. From there, try building a simple command-line tool or a single-view SwiftUI app to get comfortable with the language's flow. The official [Swift documentation](https://docs.swift.org) is excellent, and the community on the Swift Forums is welcoming to newcomers.
